### 🚀 Featured Project

**🔍 Digital Payment Fraud Detection using Machine Learning**
An end-to-end ML pipeline built on the PaySim dataset to detect fraudulent digital transactions.
- Handled extreme class imbalance (fraud = 0.13% of data) via balanced sampling
- Engineered balance-change features that became the top fraud predictors
- Compared 4 classification models; **Random Forest** selected as best (98% precision, 99.5% recall on fraud class)
- Deployed as a **live interactive web app** for real-time transaction risk scoring
